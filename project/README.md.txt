# Space Invaders – Java OOP Game 🎮

A single-level desktop game built in Java using Swing GUI. Developed as a group project to apply Object-Oriented Programming concepts.

## 🚀 Features
- Spaceship controlled by keyboard
- Falling asteroids with hit detection
- Game Over after 3 misses or 1 collision
- Score tracker and name validation
- Fully responsive layout on screen resize
- Main menu with scoreboard file viewer

## 📂 Technologies Used
- Java
- Java Swing
- Object-Oriented Programming
